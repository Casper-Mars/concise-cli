## 快速入门

* 查看帮助

```shell
~: concise-cli -help
```

* 初始化项目

```shell
~: concise-cli-${os}-{architecture}-${version} -m dao -p 1.0.1
```

> 命令执行成功后，会在执行的目录中看到多出一个dao目录，然后打开idea/eclipse打开(导入)即可。
